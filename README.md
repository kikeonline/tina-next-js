# Next.js + TinaCMS + StrapiCMS

## How to use

Install dependencies and run the example:

```bash
yarn install
yarn dev
```
Should be up and running on [http://localhost:3000]


## Includes
- TailwindCSS
- Typescript (Loose)
- PostCSS Autoprefixer
